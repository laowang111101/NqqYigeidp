# 网络舆情分析系统 python1053

## 项目概述

网络舆情分析系统是一款基于Python开发的，面向舆情监控管理人员设计的多用户支持系统。该系统通过数据分析，直观展示用户言论，并提供用户信息管理、文本分析及对比算法等多种功能。

## 技术栈

### 后端
- 开发语言：Python 3.7.7
- 框架：Django, Flask
- 数据库：MySQL 5.7+
- 数据库工具：Navicat 11+
- 开发软件：PyCharm

### 前端
- Node.js
- Vue.js
- HTML

## 功能模块

### 系统首页
- 系统公告与新闻展示
- 用户信息展示
- 系统数据统计概览

### 登录注册
- 用户注册与登录
- 表单验证流程

### 文本分析
- 文本历史输入记录
- 新文本信息分析

### 文本管理
- 关键词与结果查看
- 管理分析记录

### 个人信息查看
- 用户基础信息
- 账户登录记录

### 对比分析模块
- 算法分析结果可视化
- 柱状图与饼图展示

### 注册用户管理模块
- 用户信息查看与管理
- 用户权限设置
- 用户数据更新与删除

## 系统角色

- 普通用户：使用文本分析、个人信息管理等功能。
- 管理员：管理系统用户，处理用户信息，设置权限。

## 运行环境

- 后端：Python 3.7.7, Django, Flask, MySQL 5.7+
- 前端：现代浏览器

project/
